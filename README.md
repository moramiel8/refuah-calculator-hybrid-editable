# Refuah Styled Calculator + v6.1 Editable Source

This package contains only the calculator app (styled UI/animations) and v6.1 editable JS/JSON source files.

## What is included
- Calculator-only React app (no community/admin/profile modules)
- v6.1 source files under `v61-source/js` and `v61-source/json`
- `v61-source/duv.bat` for your familiar Windows workflow

## Run
npm install
npm run dev

## Build
npm run build

## Notes
- UI/animations come from Refuah calculator shell.
- v6.1 editable source files are included for maintenance and formula/config reference.
